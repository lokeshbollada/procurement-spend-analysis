# Procurement Spend & Supplier Performance Dashboard

This project analyzes procurement spending data using SQL Server and Power BI to understand how spend is distributed across suppliers, categories, and time.

## Tools Used
- SQL Server
- Power BI

## What I Did
- Created a procurement dataset in SQL Server
- Wrote SQL queries to aggregate spend by supplier, category, and month
- Imported the data into Power BI
- Built an interactive dashboard with KPIs and slicers

## Key KPIs
- Total Procurement Spend
- Top Supplier by Spend
- Active Suppliers
- Average Supplier Rating

## Insights
- Identifies high-spend suppliers
- Highlights major cost categories
- Shows monthly procurement spend trends
